# Frontend Rebranding — Nandex

Frontend estático criado para modernização visual do site Nandex Informática.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Estrutura

```text
nandex-rebrand-frontend/
├── index.html
├── servicos.html
├── contato.html
├── MAPEAMENTO_REBRANDING_NANDEX.md
└── assets/
    ├── css/style.css
    ├── js/main.js
    └── img/logo-nandex-rebrand.svg
```

## Personalização rápida

As cores principais ficam no início de `assets/css/style.css`:

```css
:root {
  --color-primary: #0ea5e9;
  --color-secondary: #0f172a;
}
```

## Importante

Este material é uma recriação rebrandável baseada no mapeamento público do site. Não copia o tema WordPress, scripts, CSS proprietário ou assets originais. Para publicação final, valide autorização, identidade visual, textos, fotos, telefone e WhatsApp do cliente.
